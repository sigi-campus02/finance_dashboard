API Dokumentation

So richtest du den API Key auf Render ein:
  1. Melde dich bei Bitpanda an
  2. Gehe zu: Einstellungen → API → Neue API erstellen
  3. Wähle nur Lese-Rechte (Wallets & Transaktionen anzeigen)
  4. Kopiere den generierten API Key
  5. Gehe zu deinem Render Dashboard
  6. Wähle deinen Service → Environment
  7. Klicke Add Environment Variable
  8. Key: BITPANDA_API_KEY
  9. Value: [Dein API Key]
  10. Speichern und Service neu starten
